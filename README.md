# holy-js-app

Это код тестового приложения [Tramvai](https://tramvai.dev/) для демонстрации работы с child app.

## Установка зависимостей

Перед началом работы установи зависимости с помощью `yarn`

## Запуск приложения

- `npm start` - запустит сервер в режиме разработки
- `npm build` - прод сборка приложения

## Структура приложения

* `src` - тут лежат исходники
  * `index.ts` - входной файл приложения
  * `postcss.js` - postcss конфиг
  * `routes` - страницы
  * `shared` - переиспользуемые модули приложения
